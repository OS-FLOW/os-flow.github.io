# OVERSEEN Abschiedsseite – Animiert & Responsiv

Dies ist die finale Abschiedsseite von OVERSEEN mit animiertem "GG"-Logo und responsivem Design.

## GitHub Pages Anleitung

1. Lade alle Dateien (einschließlich Bild `AdobeStock_403510827.jpeg`) in ein neues GitHub Repository hoch.
2. Gehe in **Settings → Pages**.
3. Wähle als Quelle den Branch `main` und den Ordner `/ (root)`.
4. Nach wenigen Minuten ist die Seite live unter `https://<dein-user>.github.io/<repo-name>/` erreichbar.
